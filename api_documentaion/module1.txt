## Events Routes
### Get All Events

- **Endpoint:** `GET /events`
- example url : http://localhost:8080/api/events

### Update Event Status (Read/Unread)

- **Endpoint:** `PUT /events/updateStatus/{id}/{status}`
- example url : http://localhost:8080/api/events/updateStatus/1/Read


## Tickets Routes
### Get All Tickets

- **Endpoint:** `GET /tickets`
- example url : http://localhost:8080/api/tickets


### Store a New Ticket

- **Endpoint:** `POST /tickets/store`
- example url : http://localhost:8080/api/tickets/store

**Parameters:**
- `type` (string): The type of the ticket. Must be either "Opening Ceremony Tickets" or "Closing Ceremony Tickets".
- `name` (string): The name associated with the ticket.

**Note:**
- the seat will automatically assign
